## Описание

Проект Kittygram — веб-приложение для публикации и просмотра котов, их достижений и характеристик. В рамках данной работы реализована система пользовательских предпочтений:

- ❤️ Постановка и снятие лайков
- ⭐ Персональный список избранного
- 📊 Рейтинг котов по количеству лайков
- 🔍 Расширение эндпоинтов котов полями `likes_count` и `is_liked`

---

## Функциональность

| Функция | Описание |
|---------|----------|
| Лайк кота | Авторизованный пользователь может поставить/убрать лайк |
| Избранное | Лайкнутый кот автоматически добавляется в избранное |
| Рейтинг | Просмотр топа котов по количеству лайков |
| Проверка статуса | Просмотр количества лайков и статуса лайка для каждого кота |

## Запуск проекта

### Предварительные требования

- Python 3.13+
- Docker (опционально)
- Git

### Локальный запуск (без Docker)

```bash
# Клонирование репозитория
git clone https://github.com/leravred2005-a11y/kittygram_backend.git
cd kittygram_backend

# Создание и активация виртуального окружения
python3 -m venv venv
source venv/bin/activate  # для Windows: venv\Scripts\activate

# Установка зависимостей
pip install -r requirements.txt

# Применение миграций
python manage.py migrate

# Создание суперпользователя
python manage.py createsuperuser

# Запуск сервера
python manage.py runserver
